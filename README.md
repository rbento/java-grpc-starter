# java-gradle-starter

Simple starter for Java projects with Gradle and Version Catalogs


### Requirements  


- [Gradle](https://gradle.org/releases/) 8.1.1
- [JDK 17](https://www.azul.com/downloads/?package=jdk#zulu) from Azul


### Build

```bash
./gradlew clean build
```

### Run

```bash
./gradlew :app:run
```
